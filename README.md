# Smart-Pest-Detection-System-for-Tobacco-Crops
# AI-Powered Tobacco Pest Early Warning and Economic Loss Prediction System

## Overview

The AI-Powered Tobacco Pest Early Warning and Economic Loss Prediction System is a web-based application designed to help tobacco farmers identify pests, predict future pest outbreaks, estimate crop losses, and receive treatment recommendations.

The system combines Computer Vision, Machine Learning, Weather Analytics, and Web Technologies to provide intelligent decision support for tobacco cultivation.

## Problem Statement

Pest infestations are one of the major reasons for reduced tobacco crop yield. Traditional pest monitoring methods are time-consuming and often identify problems only after significant damage has occurred.

This project aims to:

- Detect pests from tobacco plant images.
- Estimate infestation severity.
- Predict potential pest outbreaks using environmental conditions.
- Calculate expected economic losses.
- Recommend preventive and corrective measures.

## Objectives

- Develop an AI-based pest detection system.
- Provide real-time pest classification.
- Estimate infestation severity levels.
- Predict future pest outbreaks.
- Generate economic loss predictions.
- Offer pesticide and prevention recommendations.
- Maintain prediction history through a web dashboard.
